# uni-app-vue-template
> 基于 Vite + Vue3 + Pinia + uni-app H5与小程序开发模版

## support
- Vue3
- Vite
- Pinia
- Typescript
- SCSS
- 封装 uni.request
- 集成 Eslint + Stylelint + Prettier 来规范和格式化代码
- 集成 Mock 辅助开发
- 集成 uni-ui


## usage

```bash
git clone https://github.com/ueumd/uni-vue-template.git

pnpm i

pnpm run dev:h5
pnpm run dev:mp-weixin
```

## Test
目前仅测试了微信小程序和H5平台，其他平台需要自行测试
